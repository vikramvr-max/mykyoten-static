日本の永住権申請条件を徹底解説。10年標準ルート、高度専門職の3年・1年ルート、必要書類、却下理由、審査通過のための準備方法まで網羅。

永住許可は在留資格の制約から完全に解放される在留資格である。 **永住権取得後に変わること：** | 項目 | 変化 | |---|---| | 就労制限 | 撤廃（業種・時間・雇用形態を問わない） | | 在留期限 | 無期限（在留カードの更新は7年ごとに必要） | | ビザ更新 | 不要 | | 職業選択 | 完全自由 | | 副業・起業 | 制限なし | **永住権で変わらないこと：** - 日本国籍の取得（帰化ではない） - 選挙権 - 引き続き外国人として登録・在留カードの保持が必要

**経路1：10年標準ルート** 引き続き10年以上日本に在留し、うち5年以上就労ビザで在留していること。 **経路2：高度専門職ファストトラック** 高度人材ポイント計算で一定の点数を継続的に満たしている場合。 - 70点以上：3年の継続在留後に申請可能 - 80点以上：1年の継続在留後に申請可能 **経路3：配偶者ルート** 日本人または永住者と婚姻している場合。 - 婚姻後引き続き3年以上在留し、かつ引き続き1年以上「日本人の配偶者等」の在留資格を保有 **経路4：長期定住者（5年ルート）** 「長期定住者」の在留資格で引き続き5年以上在留。 **経路5：日系人ルート** 日本の国籍を以前に有していた者の子孫。

最も一般的な申請経路。以下のすべてを満たす必要がある。 **必要条件：** - 引き続き10年以上の日本在留（学生ビザ期間も算入可） - うち5年以上が就労ビザ（「技術・人文知識・国際業務」等）での在留 - 直近5年間に3か月以上の出国がないこと - 安定した収入と独立した生計能力 - 税金・社会保険・年金の完全な納付記録 - 犯罪歴なし・罰金等なし **「引き続き」の解釈：** 出国がある場合、年間合計180日以上の出国があると「引き続き」の継続性が問われる場合がある。長期出張が多い場合は事前に確認を。

ポイント制評価で高度人材ポイントが規定値以上ある場合、大幅に短縮されたルートで永住権を申請できる。 **ポイント計算の主な要素：** | 要素 | 配点例 | |---|---| | 年収（700万〜1000万円以上） | +40〜+80 | | 学歴（博士・修士・学士） | +30/+20/+10 | | 職歴（10年以上） | +20 | | 年齢（29歳以下） | +15 | | 日本語能力（N1） | +15 | | 在籍企業の規模・評価 | +10 | **重要：高度専門職ビザを持っていなくても申請できる** 申請時点でポイント要件を満たしていることが確認できれば、現在の在留資格が技術・人文知識・国際業務であっても申請可能。ポイント計算は MyKyoten の HSP ポイント計算ツールで確認できる。

「資格を満たしたらすぐ申請」は必ずしも正解ではない。 **申請前に確認すべきチェックリスト：** - 直近2年間の住民税：完全納付済みか - 直近2年間の年金保険料：完全納付済みか（免除・猶予の場合は説明資料が必要） - 直近2年間の健康保険料：完全納付済みか - 所得税の確定申告：すべての年分を申告済みか - 日本在留期間中の出入国記録：長期出国がないか **申請タイミングの実務的な考え方：** 税金・年金の記録に不備がある状態で申請しても却下される可能性が高い。在留期限が迫っている場合を除き、記録を完全に整えてから申請する方が合理的。

永住権審査において財務コンプライアンスの記録は最重要要素である。 **審査官が確認する主な財務記録：** 1. 直近5年分の課税証明書（所得証明書） 2. 直近2年分の住民税納税証明書 3. 年金保険料の納付記録（ねんきん定期便・ねんきんネット） 4. 健康保険料の領収書または証明書 5. 在職証明書・直近3か月分の給与明細 **収入の目安：** 明確な規定はないが、単身世帯で年収300万円以上、家族帯同の場合はそれ以上が実務上の目安とされることが多い。

住民税は永住権審査で最も見落とされやすい財務コンプライアンス要素である。 **仕組みの再確認：** - 毎年6月に前年収入に基づく税額が確定 - 会社員は給与から天引き（特別徴収） - 退職・転職・フリーランス転向後は自分で納付（普通徴収） **申請で問題になるケース：** - 転職時に旧職場からの特別徴収が止まり、普通徴収の通知を見落として未納になる - 退職後に自ら納付手続きをしなかった - 引越後に納税通知書が旧住所に届いて気付かなかった **確認方法：** 市区町村役場で「住民税の納税証明書（未納のない証明）」を取得し、すべての年分の記録を確認すること。

**永住権申請における年金要件：** 原則として直近2年間の年金保険料の完全納付が必要。 **ねんきんネットで確認できること：** - 月別の加入・納付状況 - 未納・免除・猶予の期間 - 将来の年金受給見込み額 **免除・猶予期間がある場合：** 正当な理由（収入減少・失業等）による申請済みの免除・猶予は、申請書内での説明があれば必ずしも致命的ではない。ただし無届けの未納は問題になる。 **厚生年金（会社員）の場合：** 給与天引きのため個人が意識しにくいが、転職時の空白期間や会社側の手続き遅延による未加入期間に注意。

高度人材ポイントは申請時点の状況だけでなく、過去の経歴・資格も含めて計算される。 **ポイントを増やすために検討すべき要素：** - 日本語能力試験（JLPT）の受験（N2で+10点、N1で+15点） - 日本の大学・大学院での学位取得（+10点） - 特許取得（+25点） - 在籍企業の規模・上場状況による加算 **70点と80点の違い：** 70点以上：高度人材として認定された日から3年後に永住権申請可能 80点以上：高度人材として認定された日から1年後に永住権申請可能 ポイントのカウント開始日は重要。既に高度人材の要件を満たしている場合は、早めに「高度専門職」の認定を受けてカウントを開始させることが戦略的に重要。

永住権申請は単なる書類提出ではなく、「この人に永住権を与えても問題ない」という説得的な記録の提示である。 **審査官の視点から考える準備：** 1. 日本社会への統合度（語学・地域活動・長期在留意思） 2. 財務的安定性（安定収入・資産・負債管理） 3. コンプライアンス記録（税金・年金・住民税・入管届出） 4. 将来の在留継続意思（家族状況・雇用状況） **「加点」になる要素：** - JLPT N2以上の日本語能力 - 日本人または永住者との婚姻 - 日本生まれの子どもの存在 - 長期の安定就労歴

**最も多い却下理由：** 1. **税金・年金・住民税の未納（最多）** 一部でも未納期間があると却下される場合が多い。 2. **在留期限が最長期間でない** 申請時点で保有している在留期限が最長（3年または5年）でない場合、審査が厳しくなる。 3. **在留歴の空白・入国記録の問題** 申請前の在留に空白期間がある、または長期出国記録がある。 4. **収入・資産の不安定性** 直近の転職・収入減少・失業歴がある。 5. **婚姻関係の実態がない（配偶者ルートの場合）** 書類上は婚姻しているが同居実態がない。 6. **書類の不備・不足** 必要書類が揃っていない、または説明が不十分。

永住権申請の却下は終わりではない。理由を分析し、不足している要件を補って再申請することが可能。 **却下後の対応手順：** 1. 却下通知書の内容を確認し、不許可の理由を特定する 2. 不足・不備のある要件を補う計画を立てる 3. 通常は却下から6か月〜1年後を目安に再申請を検討 **弁護士・行政書士への相談：** 複雑なケース（長期出国歴・未納記録・離婚歴など）は専門家への相談が有効。特に行政書士（入管法専門）への相談が一般的。

**永住権（えいじゅうけん）** - 外国籍を保持したまま日本に無期限在留できる - 外国のパスポートを引き続き使用できる - 5年ごとの在留カード更新が必要 - 10年以上在留しなくなった場合は失効リスク **帰化（きか）** - 日本国籍を取得し日本人になる - 外国籍は原則放棄（二重国籍は原則不可） - 在留カードは不要 - 選挙権・公務員への任用など市民権が付与される **選択の考え方：** 外国籍（パスポート）の保持が重要な場合は永住権。日本社会への完全な統合と権利の付与を優先する場合は帰化を検討。

永住権は一度取得すれば永久に保持できるわけではない。 **失効・取消リスクがある状況：** - 日本から出国し、**5年以上**帰国しない（再入国許可の有効期限切れ） - 重大な刑事犯罪による有罪判決 - 虚偽の申請書類による取得 **在留カードの更新：** 永住者の在留カードは7年ごとの更新が必要。更新を怠っても永住権自体は失効しないが、在留カードなしでの活動は制限される。 **長期出国の場合：** 1年以上日本を離れる場合は「再入国許可」の取得が必要。永住者の場合は最長5年の再入国許可が取得できる。

**財務コンプライアンス（申請の1〜2年前から）：** □ 住民税：直近2年分の完全納付確認 □ 年金保険料：直近2年分の完全納付確認（ねんきんネットで確認） □ 健康保険料：直近2年分の完全納付確認 □ 所得税確定申告：すべての年分の申告完了 **在留状況の確認：** □ 引き続き10年以上（または経路別の必要期間）の在留確認 □ うち就労ビザでの在留期間5年以上の確認 □ 出入国記録の確認（長期出国の有無） □ 現在の在留期限が最長期間であることの確認 **書類の準備（申請3か月前から）：** □ 直近5年分の課税証明書（所得証明書） □ 直近2年分の住民税納税証明書 □ 年金加入・納付記録の証明 □ 在職証明書・給与明細3か月分 □ 住民票（世帯全員） □ パスポート（全ページのコピー） □ 在留カード（表裏コピー）

日本の永住許可申請は書類を正しく揃えることではなく、入管が「否」と言えない記録を積み上げることである。初回申請で承認される人と却下される人の差は、資格の有無ではなく準備の質にある。
