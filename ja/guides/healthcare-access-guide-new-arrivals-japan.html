日本到着後最初の数週間に医療緊急事態が発生した場合、必要な手続きが完了していないと、高額な医療費負担を抱える可能性があります。

**先週日本へ到着したばかり。会社からは「市役所の登録には時間がかかる」と言われている。そんな中、突然体調を崩した。**

十分な準備がない場合、一般的なクリニック受診でも15,000〜25,000円かかります。救急外来なら50,000〜100,000円。手術になれば数十万円規模になることもあります。

日本の健康保険制度は世界的に見ても優れていますが、「住民登録完了後」に初めて本格的に機能します。ここが多くの新規来日者が誤解するポイントです。

日本に到着した瞬間から自動的に健康保険が適用されるわけではありません。

1. 日本国内に住所を確定する 2. 市役所で住民登録（住民票作成）を行う 3. 国民健康保険へ加入する、または会社の社会保険へ登録される 4. 保険証または資格確認情報を受け取る

会社によっては、入社日まで住所登録を待つよう指示する、社宅入居まで正式住所が決まらない、社会保険登録に数日〜数週間かかる、人事処理完了まで保険情報が有効化されないなどの事情があります。

つまり、到着から1〜3週間程度の空白期間が発生するケースが珍しくありません。しかし病気やケガは、その空白期間を待ってくれません。

- 発熱外来＋検査：15,000〜30,000円 - 点滴治療：20,000〜40,000円 - CTスキャン：30,000〜60,000円 - 救急搬送＋ER処置：50,000〜150,000円 - 入院：1日20,000〜80,000円

はい。住民票がまだなくても、日本で医療を受けることは可能です。

ただし、「保険なしで受診可能」と「保険適用価格で受診可能」は別問題です。

- パスポート - 在留カード - 仮住所情報 - 会社の雇用証明 - 健康保険加入予定を示す書類

ただし重要なのは、保険資格がまだ確認できない場合、多くの病院は「いったん全額請求」する点です。後から保険加入が確認できれば、一部返金を受けられるケースもあります。しかし病院によって対応が違い、返金手続きは日本語のみの場合が多く、数週間〜数か月かかることがあります。

**住民票登録後、できる限り早く健康保険加入を完了することが最重要です。**

新規来日者が最も混乱するポイントの一つが、「どの保険制度に入るべきなのか」です。

- 会社の社会保険（健康保険・厚生年金） - 国民健康保険（国保）

会社員の場合、多くは会社の社会保険へ加入します。しかし問題は、**入社初日から完全に保険が有効になるとは限らない**ことです。

会社によっては人事処理に時間がかかる、社会保険登録が遅れる、保険証発行まで数週間かかる、デジタル登録反映が遅いなどがあります。

つまり、正式には会社保険へ加入予定でも、一時的に無保険状態になるケースがあります。

自営業・フリーランス・留学生・無職状態の人は通常、国民健康保険へ加入します。国保は市役所で加入できます。

多くの自治体では、住民登録と同日に仮保険証、資格確認書、一時加入証明を発行してくれます。これがあれば、多くの病院で通常の3割負担を受けられる可能性があります。

**現実確認** 実際には、「会社がやってくれると思っていた」という理由で数週間放置する外国人が非常に多いです。しかしその期間中に病気になると、全額自己負担リスクがあります。

保険加入前・加入直後の時期は、通常より多くの書類を持参したほうが安全です。

- パスポート - 在留カード - マイナンバーカード（あれば） - 健康保険証または資格確認書 - 会社の雇用証明 - 日本の電話番号 - 日本の住所情報 - 現金またはクレジットカード

大病院ではデジタル資格確認やマイナ保険証確認に慣れていることが多いです。一方、小規模クリニックでは紙の保険証を要求、初診時に全額請求、英語対応なしというケースがあります。

✓ パスポート持参 ✓ 在留カード持参 ✓ 保険関連書類持参 ✓ 現金を十分準備 ✓ 病院へ事前電話確認 ✓ 英語対応有無確認 ✓ 症状を日本語メモにする

マイナンバーカードを健康保険利用登録済みなら、多くの病院で「マイナ保険証」として使えます。対応病院ではカード読み取り、デジタル保険確認、自動資格確認が可能です。

ただし注意点があります。新規来日直後は、まだマイナンバーカード未発行、健康保険情報未連携、システム反映待ちのケースが非常に多いです。

つまり、カードを持っていても、まだ医療システム上で有効化されていない可能性があります。

また、小規模クリニックではマイナ保険証未対応、機械故障、システム停止もあります。そのため、マイナンバーカードだけに依存するのは危険です。

**注意事項** 「マイナ保険証があるから大丈夫」と思い込み、他書類を持参しない外国人がいます。しかしシステム障害時には確認不能になり、一時的全額請求されるケースがあります。

健康保険加入前後のタイミングによっては、一時的に10割負担、後日保険資格確認、差額返金という流れになることがあります。

例えば、4月1日に住民登録、4月3日に病院受診（まだ保険証なし）、4月5日に国保加入完了の場合、4月3日時点で保険資格が有効扱いになれば、後日返金対象になる可能性があります。

通常必要になるのは、領収書原本、診療明細、保険証、振込先口座情報です。返金まで数週間〜数か月かかることもあります。

**現実確認** 実際には、返金制度を知らずに高額医療費をそのまま負担してしまう外国人が非常に多いです。

日本では、救急車自体は原則無料です。しかし問題は、その後の病院費用です。

救急搬送後はER診察、検査、点滴、CT、入院などが発生します。保険未加入だと、これらが全額自己負担になります。特に夜間・休日救急では費用が高くなりやすいです。

一般的には、軽症ERで20,000〜50,000円、精密検査ありで50,000〜150,000円、入院でさらに追加になります。

しかし重要なのは、**緊急時に「保険証がないから治療拒否」は基本的にされない**ことです。命に関わる状況では、まず治療が優先されます。ただし、後日請求は来ます。

**注意事項** 日本の病院は請求を無視することはありません。未払い放置は信用問題・在留関連問題につながる可能性があります。

最優先は**住民登録 → 健康保険加入**をできるだけ早く終わらせることです。

特に慢性疾患がある、処方薬継続が必要、小さい子どもがいる、妊娠中、高ストレス状態の場合は重要です。

1. 日本到着 2. 住所確定 3. 市役所で住民登録 4. 同日に国民健康保険確認 5. マイナンバー関連確認 6. 最寄り病院を事前確認

✓ 住所を確定する ✓ 市役所予約・確認 ✓ 住民登録完了 ✓ 国保加入確認 ✓ 保険証・資格確認書取得 ✓ マイナ保険証設定 ✓ 最寄り病院検索 ✓ 英語対応病院確認

**旅行保険だけで大丈夫ですか？** 短期的には役立ちますが、日本の公的健康保険加入義務の代替にはなりません。

**会社の社会保険予定なら国保加入しなくていいですか？** ケースによります。会社保険有効化前に空白期間がある場合、国保加入が必要になることがあります。

**住民登録前でも救急病院へ行けますか？** はい。緊急医療は受けられます。ただし保険未確認なら全額自己負担になる場合があります。
