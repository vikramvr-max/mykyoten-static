就労ビザ保持中に仕事を失った場合の対処法。14日以内届出義務、3か月の実務ガイドライン、雇用保険の受給資格、ビザ更新リスクを詳しく解説。

「ビザ（在留資格）」と「就労許可」は別物である。多くの人がこの点を誤解している。 在留カードに記載された有効期限は、離職によって変更されることはない。ただし、在留資格の「活動」要件は維持する必要がある。技術・人文知識・国際業務（通称：技人国）の場合、対象となる活動（エンジニアリング、翻訳、経営管理など）を行う意思を示すことが求められる。 **重要な区別：** | 項目 | 内容 | |---|---| | 在留期限 | 離職後も変わらず有効 | | 就労許可 | 在留資格の種類による | | 入管への届出義務 | 離職日から14日以内（法的義務） | | 実務上の猶予期間 | 入管ガイドラインによる3か月の目安 |

入管法第19条の16により、中長期在留者は職場を失った日から14日以内に入管局または地方出入国在留管理局へ届出を行う義務がある。 **届出方法：** 1. 最寄りの出入国在留管理局窓口への直接申請 2. 郵送による届出 3. 法務省の在留申請オンラインシステムを通じたオンライン届出 **届出に必要な情報：** - 在留カード番号 - 旧勤務先の名称・所在地 - 離職年月日 この届出は更新申請や永住権申請の審査時に入管が確認するコンプライアンス記録に残る。届出を怠った場合の即時ペナルティはないが、将来の申請に影響する可能性がある。 **ネンキン・健康保険（ケンコウホケン）についての注意：** 会社員から国民健康保険・国民年金への切替手続きも、退職後14日以内（健康保険は20日以内）に市区町村窓口で行う必要がある。

「3か月以内に再就職しなければならない」という話を耳にしたことがあるかもしれない。これは法律ではなく、入管の内部ガイドラインに基づく実務上の目安である。 **実際のリスクの発生タイミング：** 離職直後ではなく、次回のビザ更新申請時に審査官が就労状況を確認する時点でリスクが顕在化する。 **更新申請時に審査される主な要素：** - 離職期間の長さ - 離職理由（会社都合か自己都合か） - 現在の就労状況または具体的な内定の有無 - 雇用保険受給など公的手続きを適切に行っているか 雇用保険を受給しながら積極的に求職活動をしている場合は、更新審査においても正当な理由として認められやすい。

**優先度順の手続き一覧：** 1. **入管への離職届出**（14日以内・法的義務） - オンライン、郵送、または窓口で届出 2. **国民健康保険への加入**（退職後20日以内） - 市区町村窓口で手続き - 旧職場の健康保険資格喪失証明書が必要 3. **国民年金への切替**（退職後14日以内） - 市区町村窓口で手続き - 収入状況によっては保険料免除申請が可能 4. **ハローワークへの雇用保険受給申請** - 雇用保険加入期間が6か月以上あれば受給資格あり - 在留資格の種類を問わず申請可能 5. **在留資格の確認と対応方針の検討** - 現在の在留期限の確認 - 特定活動・求職活動への変更が必要かどうかの判断

日本で就労していた外国人でも、以下の条件を満たせば雇用保険の給付を受けることができる。 **受給資格の基本条件：** - 離職前の2年間で被保険者期間が通算12か月以上（特定受給資格者・特定理由離職者は6か月以上） - 積極的に求職活動を行っている - 就労可能な在留資格を保有している **給付額の目安：** 賃金日額の50〜80%（年齢・賃金水準により異なる） **給付期間：** 被保険者期間・年齢・離職理由によって90〜360日 ハローワークでの手続き時は在留カードの提示が必要。在留資格の種類と就労可否について確認される場合がある。

離職中にビザ更新時期が到来した場合、審査は厳しくなる可能性がある。ただし、適切に対処していれば承認が得られるケースも多い。 **更新審査で不利になる要素：** - 届出なしの長期離職 - 雇用保険や国民健康保険など公的手続きを行っていない - 具体的な求職活動の実績がない - ビザの在留活動と異なる業種での求職 **更新審査で有利になる要素：** - 14日以内の届出を適切に実施済み - 雇用保険受給中など公的手続きを完了 - 職務経歴書・求職活動記録の保持 - 内定書または採用予定証明書の提出

離職後に見落とされやすいのが住民税である。前年の収入に基づいて翌年6月頃に請求書が届く仕組みのため、退職後も支払い義務が継続する。 **仕組みの概要：** - 毎年6月に前年収入に基づく住民税額が確定 - 在職中は毎月の給与から天引き（特別徴収） - 退職後は自身で納付書を使って支払う（普通徴収） **退職後の支払い方法：** - 市区町村から送付される納付書を使用 - 銀行・コンビニ・PayPayなどで支払い可能 住民税の未払いはビザ更新・永住権申請の審査に影響するため、忘れずに納付すること。

**選択肢1：同業種での再就職** 現在の在留資格の範囲内で転職活動を行う。在留期限に余裕があれば、3か月程度は通常問題なく求職活動を継続できる。 **選択肢2：特定活動（求職活動）への変更** 現在の在留資格のままでは継続が難しい場合、就職活動に特化した「特定活動」への変更申請ができる。指定書に定められた条件のもとで就労可能。 **選択肢3：在留資格の変更** キャリアチェンジを検討している場合、在留資格変更許可申請が必要。審査には書類準備と時間がかかるため、早期に検討を開始すること。 **選択肢4：帰国・再入国** 一時帰国する場合も、在留期限内であれば再入国許可（または特別再入国許可）を取得した上で出国すること。

**失敗1：14日以内の届出を忘れる** 対策：退職日の翌日からカウント。オンライン届出で即日完了できる。 **失敗2：健康保険・年金の切替を放置する** 対策：退職後は会社加入の保険から自動的に外れる。市区町村で国保・国年の加入手続きを速やかに行う。 **失敗3：「3か月あるから大丈夫」と過信する** 対策：3か月はあくまで目安。更新申請時の審査官の判断次第であることを認識し、求職活動の記録を残しておく。 **失敗4：住民税の請求を無視する** 対策：退職後も前年収入に基づく住民税が届く。未払いは入管の税金コンプライアンス確認に引っかかる。 **失敗5：ビザ期限を確認せずに長期化する** 対策：離職後は在留期限をすぐに確認。残余期間が短い場合は早急に更新申請または在留資格変更の準備を始める。

就労ビザの有効期限は、離職によって自動的に取り消されることはない。ただし、入管への14日以内の届出義務、実務上の3か月ガイドライン、そしてビザ更新審査での雇用状況確認という3つの重要な課題に対応する必要がある。
